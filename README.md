# millidays

This is a little web app showing Internet time (aka Swatch internet time aka .beat time) in your browser tab and compare it with your local time.

Demo: <https://millidays.devmount.com>

<img width="1291" height="563" alt="image" src="https://github.com/user-attachments/assets/8b89fc57-7df3-47ab-a1ff-bc6e74ec1233" />

## Get started

```bash
pnpm install
```

### Compile and Hot-Reload for Development

```bash
pnpm dev
```

### Type-Check, Compile and Minify for Production

```bash
pnpm build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```bash
pnpm test:unit
```

### Lint with [ESLint](https://eslint.org/)

```bash
pnpm lint
```
